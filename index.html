JasonVM

JasonVM is a virtual machine used to execute Jason code. An example Hello world application in Jason would look like this:

//HelloWorld.jason
{
    "HelloWorld": new function(String...args)
    {
        System.console.Write("Hello World!"),
        return 0
    }
}

There is no need to compile it, as it is compiled by a JIT compiler.

The Jason programming language is based on the Javascript Object Notation (and that's how it got its name "Jason"), and extends it by adding programming functionality and more data types. The language was built to simplify GUI creation.
